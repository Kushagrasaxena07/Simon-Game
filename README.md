🎮 Simon Game

A simple and interactive Simon Memory Game built using HTML, CSS, and JavaScript. The game tests the player's memory by generating a sequence of colors that must be repeated correctly to progress to higher levels.

🚀 Features

🎯 Level-based gameplay

🎨 Animated button flash effects

🧠 Memory challenge game

❌ Game over detection

📊 Displays final score

🔄 Restart by clicking anywhere on the screen

🛠️ Technologies Used

HTML5

CSS3

JavaScript (DOM Manipulation)

🎮 How to Play

Click anywhere on the screen to start the game.

Watch the color sequence shown by the game.

Repeat the sequence by clicking the colored buttons.

Each level adds a new color to the sequence.

If you click the wrong color, the game ends and your score is displayed.
